# Context

## Problem
Sari-sari store owners commonly use notebooks to track sales, inventory, and expenses. This leads to:
- Human errors
- Difficulty tracking stock levels
- Lack of financial visibility
- No proper tracking of "utang" (credit)

## Solution
The Sari-Sari Store Inventory Management System (SSIMS) is a web-based application that:
- Tracks inventory in real-time
- Records sales and expenses
- Manages customer credit (utang)
- Provides reports for better decision-making

## Users
- Store Owner (Admin)
- Cashier / Staff

## Environment
- Runs on web browser
- Can be used on low-end computers
- Mobile-friendly interface
