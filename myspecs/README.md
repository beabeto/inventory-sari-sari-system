# Sari-Sari Store Inventory Management System (SSIMS)

## Overview
A web-based system for managing inventory, sales, expenses, and customer credit (utang) in a sari-sari store.

## Features
- Inventory tracking
- Sales management
- Expense monitoring
- Utang tracking
- Reports and analytics

## Tech Stack
- Frontend: React + TypeScript
- Backend: NestJS
- Database: PostgreSQL

## Users
- Admin
- Cashier

## Goal
To replace manual recording and improve store efficiency.
